## 游戏规则介绍
每一回合每一方都可以在棋盘上放置一颗自己的棋子
可以放置在空白处，也可以选择吃掉对方比自己小的棋子、并占据对方的位置
吃掉的棋子不会归还对方

当任何一行或者一列是相同颜色时
该颜色的一方获胜

## 定义URL前缀
你可以在`vite.config.js`文件的第8行修改URL前缀，如果您不需要任何前缀，请直接删除改文件的第8行
